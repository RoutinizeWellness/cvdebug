```markdown
# CVDebug Chrome Extension Privacy Policy

**Last Updated: January 20, 2026**

## Data Collection

The CVDebug Chrome Extension does NOT collect, transmit, or store any personal data on external servers.

## Local Storage

- Your resume text is stored locally in your browser's `chrome.storage.local`
- This data never leaves your device unless you explicitly click "Optimize Resume"
- You can clear this data at any time by clicking "Clear Resume" in the extension popup

## Permissions

The extension requests the following permissions:

- **activeTab**: To read job descriptions from LinkedIn pages you visit
- **storage**: To save your resume locally on your device
- **host permissions (linkedin.com)**: To inject the match score widget on LinkedIn job pages

## Third-Party Services

When you click "Optimize Resume", the extension opens cvdebug.com with the job description pre-filled. At that point, CVDebug's main privacy policy applies.

## Analytics

We do NOT use any analytics, tracking, or telemetry in this extension.

## Changes to This Policy

We will update this policy if our data practices change. Check back periodically.

## Contact

Questions? Email privacy@cvdebug.com
```

## ğŸ“¦ Packaging for Upload

Before uploading to Chrome Web Store:

1. **Remove development files**:
   ```bash
   cd extension
   rm -rf .git* node_modules package.json README.md CHROME_WEB_STORE.md
   ```

2. **Create ZIP file**:
   ```bash
   zip -r cvdebug-extension-v1.0.0.zip . -x "*.DS_Store" -x "*.git*"
   ```

3. **Verify ZIP contents**:
   - manifest.json (required)
   - icons/ folder with all required icons
   - src/ folder with all JS/CSS files
   - popup/ folder with HTML/JS/CSS
